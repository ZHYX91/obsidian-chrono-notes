# 产品需求

## 1. 产品定位

Chrono Notes 是周期笔记工作空间，而不只是一个日期选择器。它以年、月、周日历组织日记、周记、月记、季度笔记、年记、任务、区间笔记和只读外部日程。

重写首要目标是达到旧项目功能对等，并改善一致性、可访问性、窄侧边栏体验和数据可靠性。功能对等完成前，不扩展为通用项目管理或网络日历系统。

## 2. 日历视图

- 主日历顶栏按“上一周期、周期选择器、下一周期、弹性留白、回到当前周期”组织：箭头与选择器之间使用紧凑间距，右侧动作至少保留可辨识间距，但不采用对称留白。月视图使用年/月选择器和“今天”，周视图使用 ISO 周年/周号选择器和“今天”，年视图使用年份选择器和“本月”。年、月、ISO 周年与周号弹层采用同一独立状态语法：current 只使用强调文字色和 `aria-current`，不使用圆点；selected 只使用 `2px` 内描边和 `aria-pressed`，不改变文字色或背景；focus 使用独立外描边，current 与 selected 可以共存。月份仅在所浏览年份为本年时标记本月，周号仅在所浏览 ISO 周年为当前周年时标记本周。

### 2.1 月视图

- 显示与目标月相交的完整周和周序号列，按实际边界形成 4–6 周，不显示完全属于相邻月份的整周；周序号可选择整周、打开对应周记，并以不参与排版的顶部状态层显示周记状态或任务进度。周记未配置或状态隐藏时不保留空行，周号始终相对整格居中。月主网格默认使用 `6px` gap，`≤360px` 时收紧为 `4px`；周序号轨道与七列日期之间另有稳定 spacer 留白，不使用分隔线。每个周序号入口是独立格，四角均使用与日期格一致的 `5px` 圆角，不保留轨道式的右侧直角。包含本地今日的周号使用贴格子内缘的中性 `1px` 内边框，所选周号使用位于格子边界之外的强调色 `2px` 外框；两种状态可叠加，焦点在 selected 外侧作为独立最外层显示，且均不改变背景、文字颜色或字重。
- 日期格可显示日期、调休标记、历法文本、笔记状态、任务进度、节假日和 ICS 摘要；顶部固定为左侧笔记状态/横向任务进度与右侧地区标记，日期号独占下一行并居中，日期号与后续详情保持 `3px` 有效间距。无 ICS 时不预留固定空块，内容增多时同周日格统一自然增高。包含本地今日的日期格使用贴格子内缘的中性 `1px` 内边框，选择使用位于格子边界之外的强调色 `2px` 外框，焦点再在 selected 外侧作为独立最外层显示；current、selected 和 focus 均不改变背景、文字颜色或字重，区间预览与拖放目标等临时态优先。月热力图保留四档数据背景，日期号水平和垂直居中，不渲染区间甘特，并沿用同一状态框线矩阵。月首尾的非本月日期必须整格弱化，同时保持当天、选择、区间预览、悬停和键盘焦点的完整对比度。
- 支持选择、打开日记、右键菜单、键盘导航和拖选至少两个日期创建区间笔记；反向拖选必须归一化。
- 热力图按统一统计维度显示四档强度。

### 2.2 周视图

- 顶栏分别选择 ISO 周年和周号，不复用会切换到月视图的迷你日历。周号列表按目标周年动态提供 52 或 53 周；切换周年时不存在的第 53 周收敛到最后一周。选择周年或周号后保持周视图并尽量保留当前选中日的星期偏移；周日起始只改变七日展示边界，不改变 ISO 周身份。
- 周号弹层每周使用一个至少 44 px 高的完整按钮，同行展示 `W01` 与紧凑日期范围；常规宽度三列，窄于 360 px 时两列并可纵向滚动。当前周、所选周、键盘焦点和范围文本必须可区分，并支持方向键、Home、End、Page Up/Down、数字定位、Escape 和焦点恢复。
- 聚合一周任务，并按日期、截止时间和完成状态组织。
- 只以截止日期判断逾期。
- 首版拖拽改期仅改写任务中的截止日期。
- 七日格复用月格的完整日模型，依次展示状态/调休、星期、日期、历法扩展、节假日和 ICS，并按宽、中、窄三档容器密度逐级折叠可见详情而不丢失无障碍信息。周主网格默认使用 `6px` gap，`≤360px` 时收紧为 `4px`；日期与其后详情保持 `3px` 有效间距。今日格使用贴格子内缘的中性 `1px` 内边框，选择格使用位于格子边界之外的强调色 `2px` 外框，焦点再在 selected 外侧作为独立最外层显示；三者均不改变背景、文字颜色或字重。周区间甘特使用同一组 `6px / 4px` 列间距，与七个日期格精确对齐。
- 展示日记、周记和区间笔记；区间甘特在七个日按钮之后以独立的正常流 sibling section 对齐跨列显示，不位于日格内部，也不重复列出。区间与任务的空状态使用零计数或紧凑创建入口，不叠加冗余说明段落。

### 2.3 年视图

- 右侧“本月”动作选择本地当前月份的第一天并保持年视图，以月份选择态呈现；若当前显示的不是本年，同时导航回本年并滚动到目标月份。
- 概览模式展示季度和月份入口，常规格间距为 `6px`，`≤360px` 时为 `4px`；可见状态使用不参与排版、距格顶 `4px` 的叠加层，与标签保持约 `2px` 可见间距。笔记未配置或状态隐藏时不保留空行，季度/月份标签始终相对整格居中，入口格高不因状态层移动而改变。包含本地今日的季度和月份使用贴格子内缘的中性 `1px` 内边框，选择使用位于格子边界之外的强调色 `2px` 外框，焦点再在 selected 外侧作为独立最外层显示；三者均不改变背景、文字颜色或字重。
- 热力图模式展示全年每日统计强度，日热力小格常规间距为 `4px`，`≤360px` 时为 `3px`；季度/月入口继续使用上述周期状态，日热力格保留统计背景并沿用同一套贴格子内缘的中性 `1px` current 内边框、位于格子边界之外的强调色 `2px` selected 外框，以及位于 selected 外侧的独立最外层焦点。
- 月份和季度支持选择及打开对应周期笔记。

## 3. 周期笔记

- 支持日、周、月、季度、年五种周期。
- 每种周期独立配置启用状态、文件路径 pattern 和模板。
- 同一日期语义必须用于路径生成、路径反向识别、导航和模板上下文。
- 支持内置模板变量与 Templater 适配；Templater 不可用时明确报错，不静默降级。
- 周期笔记与区间笔记分别配置创建确认；确认窗口可在本次创建同时关闭对应类别的后续确认，取消、关闭或 Escape 不得修改设置。
- 可配置较大周期笔记级联创建；一次主笔记创建只确认一次，级联项不重复确认。

## 4. 历法与节假日

### 4.1 历法扩展

功能对等版本实现两个可选历法扩展，用户按显示顺序最多选择两个：

- 中国农历：农历月日、节气与传统节日；
- 干支历：平日显示日柱，交节日显示新的月柱，并保留完整年、月、日柱信息。

公历是基础日历，不属于可选扩展。扩展使用显式静态 provider 注册表；不得运行时扫描文件或展示未实现选项。单选居中显示，双选在空间充足时等宽并排、窄栏中纵向排列，字号不因选择数量变化。

### 4.2 节假日扩展

首个功能对等版本仅支持：

- 中国大陆公共假日与调休；
- 新加坡公共假日。

历法与节假日是独立设置，不能用一个“地区模式”绑定语言、历法和节假日。
节假日地区使用三个有序槽位，设置页只列出已实现 provider；当前只有中国大陆和新加坡，因此第三槽可以为空。槽位顺序控制名称与普通假日标记顺序，补班标记始终优先。地区标记使用解析后的插件界面语言：简体/繁体中文为 `班 / 休 / 公假`，英文为 `Work / Off / PH`。

## 5. 笔记内容、任务与统计

- 笔记状态包括不存在、空文件、仅 YAML、存在正文和读取错误。
- 正文状态、预览、任务和统计必须来自同一次统一解析。
- 解析必须兼容 BOM、LF、CRLF、CR、混合换行和 YAML 结束标记位于 EOF。
- 统计支持字数、链接数、标签数和任务完成率。
- 月视图和年视图必须共享统计口径。
- 月、年热力图开关相互独立；任一当前视图启用热力图时，右侧工具组展示同一统计维度选择器、与该视图实际色阶一致的 0–4 档图例及热力图开关。关闭时只保留开关，周视图不展示热力图工具。窄容器先隐藏图例，再允许整组换行并保持右对齐。

## 6. 区间笔记

- 使用 `start` 和 `end` frontmatter 表达连续日期范围。
- 支持月视图、周视图和相关周期笔记中的轻量展示。
- 普通月视图和周视图使用跨日期列的简化甘特泳道；月热力图不渲染区间甘特。重叠区间分行、不重叠区间复用行，跨周片段保持稳定身份色。
- 支持创建、搜索、筛选和排序；关闭区间创建确认不跳过起止日期输入、路径冲突或错误反馈。
- 不扩展依赖关系、里程碑、重复区间或完整甘特编辑器。

## 7. 外部日历

- 仅导入本地 ICS 文件并只读展示。
- 不进行网络订阅、CalDAV、写回或 recurrence 展开。
- 普通宽度月格单日最多显示 3 条 ICS 摘要，超出后使用 `+N`；窄布局可折叠可见摘要，但保留完整无障碍信息。

## 8. 交互与辅助入口

- 单击选择；双击、长按、中键或 Ctrl/Cmd 单击打开笔记。
- 支持在新标签页打开、日期右键菜单、迷你日历和跳转日期。
- 主日历提供高效的年份、月份、ISO 周年和周号选择，并可从对应年、季度或月份目标打开周期笔记；具体弹层布局可以改进旧版 20 年和季度分组表格，但不得丢失这些动作。
- 周期笔记内支持上一周期、下一周期和下一个已启用的上级周期导航；前后/选日主导航必须相对编辑区严格居中，日历与上级周期入口位于独立右侧动作区，不能使中心随按钮有无漂移。移动端 Navbar 必须位于系统安全区和 Obsidian 原生视图头下方、编辑正文上方，不得覆盖系统状态栏、挖孔区域或宿主按钮，且所有既有动作仍可触达。
- 所有仅靠颜色表达的状态必须同时有形状、文本或无障碍标签。
- 主日历统一以 `is-current-period` 标记包含本地今日的日期或周期：精确日使用 `aria-current="date"`，周、月、季入口使用 `aria-current="true"`，选择语义与 current 独立。主网格的 current 使用贴格子内缘的中性 `1px` 内边框，selected 使用位于格子边界之外的强调色 `2px` 外框，focus 再位于 selected 外侧作为独立最外层；三者不改变背景、文字颜色、字重或装饰并可同时存在。区间预览/端点、任务拖放目标等临时态具有视觉优先级，但 selected 与 focus 仍保持独立。该主网格契约不直接复用于周期选择器或迷你日历；周期选择器按自身目标和当前浏览上下文独立派生 current/selected/focus，迷你日历保持既有状态样式。

## 9. 设置与语言

- 设置页使用“常规、外观与视图、周期笔记、区间笔记、集成”五个选项卡。
- 设置数据有 schema 版本、校验和迁移机制。
- 设置变化只刷新受影响的数据与界面。
- UI 支持英文、简体中文和繁体中文。
- 周期笔记设置按日、周、月、季、年独立启用；关闭的类型收起路径与模板细项，启用后可从 Vault 文件夹和 Markdown 文件建议中选择，并实时预览由统一路径规则生成且可反向识别的完整笔记路径。建议选择只写回现有路径格式或模板路径，不新增旁路配置。
- 外观与视图设置保留字体跟随 Obsidian、跟随侧栏和固定倍率，提供两个有序历法扩展槽位、“显示笔记状态”开关，以及数字或春夏秋冬季度命名。字体默认使用固定模式和 `0–20` 倍率因子（默认 `10`）；跟随侧栏使用离散容器档位，不能按视口宽度连续缩放。季度命名默认使用按界面语言本地化的数字名称，中文模式固定显示 `春 / 夏 / 秋 / 冬`，并同时作用于年视图与月选择器。开关关闭只隐藏月、周、年中的可见状态圆点或进度，不能删除 NoteIndex 数据、任务统计或 ARIA；开关开启时进度统一为顶部横向轨道，不再提供位置选择。日期格顶部右侧固定保留给地区日历标记，历法扩展不得占用。
- 任务标注保留旧版 `none | color | hole` 三种模式并默认使用 `hole`：`none` 只关闭可见任务进度，`color` 用警示色轨道明确未完成，`hole` 用空心轨道明确未完成；任何模式都不得删除 NoteIndex 中的任务统计、热力图/预览数据或 ARIA。旧版没有渲染消费者的统计点上限不迁移，月/年热力图统一固定为 0–4 级。
- 常规设置可选择是否使用 Chrono Notes 打开 Obsidian Properties 中的日期属性；默认关闭。启用时只捕获日期值旁的打开图标，按 Chrono Notes 日记路径以默认目标或 `Ctrl / Cmd` 新标签页打开；Properties 中的 Wiki/Markdown 链接与正文链接保持 Obsidian 原有行为。

## 10. 非目标

- 独立日视图；
- 网络日历同步和事件写回；
- 通用脚本模板语言；
- 完整项目管理或甘特编辑器；
- 未实现历法和地区的占位设置。
