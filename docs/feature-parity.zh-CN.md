# 功能对等清单

状态：`[x]` 表示当前实现与自动契约已具备该能力；发布级真实宿主证据单独列在文末。Chrono Notes 当前已达到旧项目的功能范围，现行产品、架构与 UX 文档优先于旧项目实现细节。

## 基础与数据

- [x] 独立插件 ID `chrono-notes`、生产构建、版本化 manifest 和三文件安装产物。
- [x] schema v15 设置迁移、严格归一化、未知字段剔除和独立默认对象。
- [x] UTF-8 BOM、LF/CRLF/CR/混合换行、EOF frontmatter、`---`/`...`、YAML 错误与资源上限解析。
- [x] NoteIndex 统一拥有笔记存在性、规范文档、预览、任务、统计和区间事实。
- [x] Vault create/modify/rename/delete、逐路径修订、生命周期、读取去重、初始 staging、错误快照和旧结果屏障。
- [x] 任务日期与区间增量投影、结构共享、no-op modify、事件归约和有界 live batch 发布。

## 周期笔记

- [x] 日、周、月、季、年五类路径生成、反向识别、打开、创建和前后导航。
- [x] 周一/周日起始、ISO 跨年周、季度与年度锚点。
- [x] 内置模板、Templater 目标周期上下文、失败回滚和明确错误。
- [x] 周期创建确认、同路径并发协调和较大周期级联。
- [x] Note Navbar 前后、选日、有效上级周期和打开日历；完整路径识别、rename 与叶生命周期。

## 日历视图与交互

- [x] 月视图按真实边界显示 4–6 个完整周、周号、跨月弱化、日记状态、节假日、ICS 和区间甘特。
- [x] 周视图显示七日详情、周记、日期任务、区间段和创建入口。
- [x] 年概览显示四季度与十二个月；年热力图显示完整公历年、闰日和季度分段窗口。
- [x] 年/月/ISO 周年/周号选择器、当前周期动作、键盘、焦点恢复和新标签打开。
- [x] 单击选择，双击或 Enter 默认打开，中键或 Ctrl/Cmd 新标签打开，触摸长按。
- [x] 日期右键打开/创建、区间入口和复制日期。
- [x] 月格拖选正向或反向日期范围并进入统一区间创建链。
- [x] 月/年热力图共享字数、链接、标签和任务完成率维度及 0–4 等级。
- [x] 月/年悬浮与焦点预览、视口定位、延迟取消和 ARIA 关联。

## 内容、任务与统计

- [x] 空文件、仅 frontmatter、正文和 note-local 解析错误的独立状态。
- [x] 统一预览、字数、链接、标签、任务总数、完成数和完成率。
- [x] Tasks emoji due/scheduled/start 日期解析与源行身份。
- [x] 周任务按七日 bucket 聚合、同任务同日去重和稳定排序。
- [x] 仅未完成 due 任务表达逾期，并区分 1–2 天与 3 天以上等级。
- [x] 任务 checkbox 原子切换、来源行定位和当前周 due 改期。
- [x] 顶部笔记状态圆点或横向任务进度；显示开关只影响可见指示器，不删除任务数据或 ARIA。

## 历法、节假日与 ICS

- [x] 中国农历、二十四节气、传统节日和干支纪日；结构日期与事件分离，支持简中、繁中和英文。
- [x] 中国大陆法定假日/调休和新加坡公共假日；覆盖状态与普通工作日分离。
- [x] 两个有序历法槽与三个有序地区槽，补班优先且同日只显示一个地区标记。
- [x] ICS Vault 文件、桌面 Vault 相对路径和桌面绝对路径只读来源。
- [x] RFC 5545 折叠/转义、全天排他结束、UTC/IANA/浮动时间、跨日与稳定排序。
- [x] recurrence 明确计数跳过、非法事件隔离、逐来源状态、并发刷新与旧结果保护。

## 区间笔记

- [x] 严格 `start`/`end` 日期或日期时间解析、同日范围、包含首尾天数和显式错误。
- [x] 确定性创建路径、确认、父目录、已有文件打开和失败边界。
- [x] 区间目录、自定义目录或整个 Vault 扫描范围。
- [x] 月/周共享泳道布局、跨周延续、稳定身份色、显示上限和可操作 `+N` 隐藏项面板。
- [x] 区间列表搜索、当前月/年相交筛选、稳定排序、创建和设置/Vault 实时刷新。

## 设置、国际化与可访问性

- [x] 常规、外观与视图、周期笔记、区间笔记、集成五个原生设置页签。
- [x] English、简体中文、繁体中文完整消息目录和 `auto` 解析。
- [x] 字体跟随 Obsidian、侧栏离散分档或固定倍率；季度名称与任务标注模式。
- [x] Vault 路径手动输入、完整预览、错误反馈、共享候选目录和 bounded top-100 模糊建议。
- [x] current、selected、focus 正交状态、非颜色状态文字、可见焦点、ARIA 名称和粗指针目标。
- [x] 首次使用引导和 Properties 日期打开日记开关。
